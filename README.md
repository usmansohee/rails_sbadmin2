# Rails dashboard with startbootstrap sb-admin2

## Using:

- Ruby 3.0.4
- Rails 6.1.7
- Bootstrap 4

## Download and Installation
```sh
cd project_name
```

```sh
bundle install
```

```sh
yarn
```

```sh
rails db:create
```

```sh
rails server
```
